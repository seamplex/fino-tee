# Fino case: parametric study of a piping tee

This repository contains the input files to perform a parametric analysis of a piping tee.

# Execution

Required packages:

 * GNU M4
 * Fino
 * Gmsh

```
./run.sh
```
 
# Plotting

Required packages:

 * Pyxplot
 * ParaView
 * Python
 * ParaView-Python
 
```
./plot.sh
```
